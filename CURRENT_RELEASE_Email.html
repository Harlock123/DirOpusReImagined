DirOpus Reimagined v0.1.21.0
A fast, cross-platform dual-panel file manager — inspired by the classic Amiga Directory Opus.

Hi! I've been building a file manager and just released a new version — thought you might like to try it. It runs on Windows, macOS, and Linux (Intel & ARM), it's free and open source, and there's nothing to install — just unzip and run.

The two-panel main window (Solarized Light theme).
What it does
  • Two folders side by side — copy, move, and compare between them, with folder tabs on each side.
  • All the file basics — copy / move / rename / delete, drag-and-drop, and delete-to-Trash (recoverable).
  • Compare & sync two folders with a click.
  • Cloud storage built in — Google Drive, OneDrive, Dropbox, Amazon S3, and 50+ more, browsed like local folders.
  • Look inside archives — open .zip, .7z, .rar, and .tar like folders.
  • Make it yours — 36 custom buttons to launch your own tools, plus bookmarks and a built-in file viewer.
  • Themes — Light/Dark, Dracula, Nord, and Solarized.
  • Fast to drive — recursive search, type-ahead jumping, and full keyboard control.
New in this release (0.1.21.0): a much more powerful batch rename — rename a whole selection at once with a live before→after preview, find&replace (with regex), sequence numbering, and case changes. It highlights name clashes before you commit, so nothing gets clobbered by surprise.
